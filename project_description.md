# Project Description: Online Food Delivery Management System

## ER Model (Text Description)

Entities:
- Customer
- Restaurant
- MenuItem
- Orders
- OrderItem
- DeliveryPartner
- Delivery
- Payment

Relationships:
- Customer places Orders
- Restaurant offers MenuItems
- Orders include OrderItems
- DeliveryPartner delivers Orders
- Payments belong to Orders

This project covers:
- Database design
- SQL DDL & DML
- Joins, constraints, aggregate functions
